## 📝 주요 콘텐츠

## 🚀 로컬 실행

```bash
# 저장소 클론
git clone https://github.com/yujeong0411/yujeong0411.github.io.git
cd yujeong0411.github.io

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```
